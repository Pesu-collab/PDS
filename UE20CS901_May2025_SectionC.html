Question 4 · Part a

Car Dataset — Filtering, Grouping & Binning

1

■ Original / Rewritten Question

Load the car dataset and perform the following:

  1. Find the number of cars with Price > $25,000, and calculate the average mileage of those cars. Write your inference. (4)
  2. Separate the data into Petrol and Diesel cars; compute and print the average price for each group. (5)
  3. Summarise the inventory by make: the average price and the maximum mileage for each brand. (3)
  4. Add a column Mileage Category classifying cars as Low / Moderate / High:
    0 < Mileage < 30,000 → Low, 30,000 < Mileage < 100,000 → Moderate, Mileage > 100,000 → High. (4)
  5. Identify the highest-priced car for every make. (4)
2

■ Problem Understanding

Pure pandas data analysis — no machine learning. Each of the five parts is one core skill:

  • (i) Filter + aggregate — keep rows by a condition, then average a column.
  • (ii) Group + mean — "separate" just means groupby("Fuel_Type").
  • (iii) Group + two summaries — one groupby producing two different stats with .agg().
  • (iv) Bin a number into labels — the textbook use of pd.cut().
  • (v) Best row per group — groupby(...).idxmax() then .loc[].

The marks reward translating the English into the right one-liner and writing a short inference (the paper's instructions say "include observations and inferences wherever appropriate").

■ Detailed Explanation (step by step)

(i) df["Price"] > 25000 makes a True/False mask; df[mask] keeps the True rows; len() counts them; ["Mileage"].mean() averages mileage for just those cars.

(ii) groupby("Fuel_Type")["Price"].mean() splits rows into a Petrol bundle and a Diesel bundle and averages Price inside each.

(iii) groupby("Make").agg(avg_price=("Price","mean"), max_mileage=("Mileage","max")) — .agg() with named outputs lets one grouping return two different stats in a tidy table.

(iv) pd.cut(col, bins=[0,30000,100000,inf], labels=[...]) drops each mileage into a bucket. Four edges → three buckets. Edge handling: by default pd.cut is right-inclusive, so a value of exactly 30,000 lands in Low and exactly 100,000 in Moderate — a clean, defensible convention for the strict-inequality wording in the question.

(v) groupby("Make")["Price"].idxmax() returns the row-label of the most expensive car within each make; feeding those labels to df.loc[...] pulls the full rows.

■ Complete Python Program

# section_4a.py 
import pandas as pd                  # tables, filtering, grouping
import numpy as np                  # np.inf for the open-ended mileage bin

df = pd.read_csv("car_data.csv")    # load CSV into a DataFrame

# (i) Cars priced above $25,000  +  their average mileage
expensive = df[df["Price"] > 25000]              # mask keeps Price>25000 rows
print("(i) Count >$25k :", len(expensive))
print("    Avg mileage :", round(expensive["Mileage"].mean(), 2))  # mean ignores NaN

# (ii) Average price for Petrol vs Diesel
print("(ii) Avg price by fuel:")
print(df.groupby("Fuel_Type")["Price"].mean().round(2))

# (iii) Per-make summary: average price AND maximum mileage
summary = df.groupby("Make").agg(
    avg_price=("Price", "mean"),                 # name -> (column, function)
    max_mileage=("Mileage", "max")
).round(2)
print("(iii) Per-make summary:")
print(summary)

# (iv) New column 'Mileage Category' via pd.cut (bin a number into labels)
df["Mileage Category"] = pd.cut(
    df["Mileage"],
    bins=[0, 30000, 100000, np.inf],             # 4 edges -> 3 buckets
    labels=["Low", "Moderate", "High"]
)
print("(iv) Category counts:")
print(df["Mileage Category"].value_counts())

# (v) Highest-priced car for every make
idx = df.groupby("Make")["Price"].idxmax()       # row-label of dearest car / make
print("(v) Top car per make:")
print(df.loc[idx, ["Make", "Model", "Year", "Price"]]
        .sort_values("Price", ascending=False))

■ Problem Understanding

A visualisation + inference section. Four of the five parts produce a chart; one is a cleaning-then-group calculation. ■ Detailed Explanation (step by step)

  • (i) Boxplot, hue=Fuel_Type — one pair of boxes per make (Petrol vs Diesel), showing median and spread side by side.
  • (ii) Clean then group — dropna(subset=["Price","Mileage"]) removes only rows missing those two; then groupby("Make")["Price"].mean().
  • (iii) Countplot, hue=Fuel_Type — bar heights show how many of each fuel per brand; the mode (s.mode()[0]) gives the most common fuel per brand.
  • (iv) lmplot, hue=Transmission — scatter of Price vs Age with a separate regression line per transmission, answering "trend with age" and "manual vs automatic" in one chart. The correlation number quantifies it.
  • (v) Countplot of Make — sorted bars; the tallest is the most-listed make.

Every figure gets title + xlabel + ylabel + colour palette + an inference.

■ Complete Python Program

# section_4b.py 
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns
sns.set_theme(style="whitegrid")

df = pd.read_csv("car_data.csv")
df["Age"] = 2025 - df["Year"]        # used by part (iv)

# (i) Boxplot: Price per Make, grouped by Fuel_Type
plt.figure(figsize=(11, 6))
sns.boxplot(data=df, x="Make", y="Price", hue="Fuel_Type", palette="Set2")
plt.title("Distribution of Car Prices by Make, grouped by Fuel Type")
plt.xlabel("Make"); plt.ylabel("Price (USD)")
plt.xticks(rotation=30, ha="right")
plt.tight_layout(); plt.show()

# (ii) Average price per make, ignoring rows missing Price OR Mileage
clean = df.dropna(subset=["Price", "Mileage"])   # drop only those NaN rows
print("(ii) Avg price by make (cleaned):")
print(clean.groupby("Make")["Price"].mean().round(2))

# (iii) Fuel-type spread across brands + most common fuel per brand
plt.figure(figsize=(11, 6))
sns.countplot(data=df, x="Make", hue="Fuel_Type", palette="Set1")
plt.title("Fuel Type Distribution across Car Brands")
plt.xlabel("Make"); plt.ylabel("Number of Cars")
plt.xticks(rotation=30, ha="right")
plt.tight_layout(); plt.show()

most_fuel = df.groupby("Make")["Fuel_Type"].agg(lambda s: s.mode()[0])  # the mode
print("(iii) Most common fuel per brand:"); print(most_fuel)

# (iv) Price vs Age, split by Transmission (scatter + trend line per group)
sns.lmplot(data=df, x="Age", y="Price", hue="Transmission",
           height=6, aspect=1.5, scatter_kws={"alpha": 0.4}, palette="Dark2")
plt.title("Car Price vs Age, by Transmission Type")
plt.show()
print("Price-vs-Age correlation:", round(df["Age"].corr(df["Price"]), 3))

# (v) Most-listed make
counts = df["Make"].value_counts()
plt.figure(figsize=(10, 6))
sns.countplot(data=df, y="Make", order=counts.index, palette="viridis")
plt.title("Number of Listings per Car Make")
plt.xlabel("Number of Cars"); plt.ylabel("Make")
plt.tight_layout(); plt.show()
print("Most listed make:", counts.idxmax(), "->", counts.max())
